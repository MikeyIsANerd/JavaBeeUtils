# JavaBeeUtils
> [!WARNING]
> JavaBee is still under development, an alpha release is in the works!

JavaBee Utils is a public library dedicated to making FTC programming even easier. 
When it was just a spark in our minds, we adored libraries like Pedro Pathing. 
Public libraries that made one of the hardest parts of FTC doable. 
Our goal is to make not just advanced techniques, but the foundations of Java in FTC, easier for rookies and veterans alike.
